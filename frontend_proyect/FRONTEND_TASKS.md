# Tareas del Frontend (Juan)

Este archivo describe exclusivamente lo que le toca desarrollar a Juan para el proyecto FinanzaFlow.

## Sprint 1

**Objetivo**

Construir la estructura inicial de la aplicación y desarrollar el módulo de registro de ingresos y gastos.

**Tareas de Juan**

- Configurar el proyecto con Vite.
- Configurar Tailwind CSS.
- Crear la estructura de carpetas.
- Implementar LocalStorage para guardar datos de forma local.
- Configurar GitHub y el control de versiones.

## Sprint 2

**Objetivo**

Desarrollar el Dashboard y el sistema de cálculos financieros.

**Tareas de Juan**

- Programar el cálculo del balance.
- Programar los reportes mensuales.
- Programar los reportes anuales.
- Integrar Chart.js para gráficas.

## Sprint 3

**Objetivo**

Finalizar la aplicación y realizar pruebas.

**Tareas de Juan**

- Programar recomendaciones financieras.
- Optimizar el rendimiento.
- Corregir errores.

## Entregables de Juan

- SPA funcional con navegación entre vistas.
- Registro de ingresos y gastos con edición y eliminación.
- Cálculos automáticos de balance.
- Reportes y estadísticas con gráficos.
- Recomendaciones y simulador de ahorro.
- Configuración básica para modo mock/backend.

## Notas

Este archivo es la referencia de las tareas ejecutadas por Juan. Las tareas del compañero se documentan en `BACKEND_TASKS.md`.
